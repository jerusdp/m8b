# m8b

A Magic Eight Ball implementation returning a random Magic Eight Ball response each time it is run.

Possible responses include:

- AsSeen
- Later
- BetterNotNow
- CannotPredict4
- Concentrate
- DoNotCount
- Certain
- Decidedly
- MostLikely
- ReplyNo
- SourcesNo
- OutlookNotGood
- OutlookGood
- ReplyHazy
- SignsYes
- VeryDoubtful
- WithoutDoubt
- Yes
- YesDefinite
- RelyOnIt
